# autd3_nim

version 2.2.1

## Run examples

- First, change directory to `examples`, and then run `build.ps1` (Windows) or
  `build.sh` (Linux, macOS) to download libraries.

  ```
  cd examples
  pwsh build.ps1
  ```

  or

  ```
  cd examples
  ./build.sh
  ```

- And then, run examples

  ```
  nimble run soem
  ```

  or

  ```
  nimble run emulator
  ```
